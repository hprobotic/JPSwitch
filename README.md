##Requirements
- iOS 8.0+
- Xcode 7.0.1

## Installation

> **Embedded frameworks require a minimum deployment target of iOS 8.**
>
> To use with a project targeting iOS 7, you must include the `JPSwitch.swift` and `JPSwitch.xib` source file directly in your project. 
>
